نشاط تفاعلي: تحدي المتغيرات

1

تحدي التصنيف

صنف المتغيرات التالية حسب نوع البيانات المناسب:

عمر الطالب
اسم المدرسة
معدل الدرجات
ناجح/راسب
2

تحدي الأخطاء

حدد الخطأ في كل سطر من الأكواد التالية:

int

age = "25";

string

name = محمد;

bool

isActive = 1;

double

price = 19,99;
3

تحدي الإكمال

أكمل الكود التالي بإضافة نوع البيانات المناسب:

// أكمل نوع البيانات المناسب

_____ studentName = "سارة";
_____ studentAge = 18;
_____ studentGPA = 3.75;
_____ isGraduated = false;
4

تحدي التطبيق

اكتب كود C# لإنشاء المتغيرات التالية:

  • متغير لتخزين درجة الحرارة (عدد عشري)
  • متغير لتخزين عدد الطلاب (عدد صحيح)
  • متغير لتخزين اسم المادة (نص)
  • متغير لتخزين حالة الدوام (منطقي)
