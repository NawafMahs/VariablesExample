المتغيرات في C#

المتغير هو مساحة في ذاكرة الحاسوب تستخدم لتخزين البيانات:

  • يمكن تغيير قيمته خلال تنفيذ البرنامج
  • له اسم يميزه عن باقي المتغيرات
  • له نوع بيانات يحدد نوع القيم التي يمكن تخزينها
  • له قيمة مخزنة فيه

🧠 تشبيه لتقريب المفهوم:

المتغير مثل صندوق له:

  • ملصق عليه (اسم المتغير)
  • نوع محدد من المحتويات يمكن وضعها فيه (نوع البيانات)
  • محتوى داخله (القيمة)

// أمثلة على إعلان المتغيرات في C#

int

age = 20; // متغير لتخزين العمر

string

name = "أحمد"; // متغير لتخزين الاسم

double

height = 175.5; // متغير لتخزين الطول

bool

isStudent = true; // متغير لتخزين حالة الطالب

💡 فكر معي:

ما هي الأشياء في حياتنا اليومية التي يمكن تمثيلها بمتغيرات؟

(مثال: درجة الحرارة، عدد الطلاب، اسم المدرسة)
