الأنواع الأساسية للمتغيرات في C#:

الأعداد الصحيحة (Integer)

int, long, short, byte

الأعداد العشرية (Floating Point)

float, double, decimal

النصوص (Text)

char, string

القيم المنطقية (Boolean)

bool (true/false)

مقارنة بين أنواع البيانات الشائعة

النوع المدى مثال
int -2.1 مليار إلى 2.1 مليار int age = 25;
double ±5.0 × 10−324 إلى ±1.7 × 10308 double price = 19.99;
string 0 إلى 2 مليار حرف string name = "محمد";
bool true أو false bool isActive = true;
